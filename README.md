# Batman Keychain – Onshape

## Project Overview

This project presents a 3D Batman keychain designed using Onshape.

The design was created as a CAD model by creating a Batman logo sketch and converting it into a 3D printable keychain using the Extrude feature. A circular hole was added to allow attaching a keychain ring.

## Requirements

- Design a Batman keychain model.

## Design Process

1. Created a sketch of the Batman logo on the Top Plane.
2. Added dimensions and sketch constraints.
3. Designed the keychain hole.
4. Used the Extrude feature to create the 3D model.
5. Exported the final model as an STL file.
